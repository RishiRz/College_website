# 🎓 College Website

A dynamic college website designed to manage student information, announcements, faculty details, and resources in one place. This project includes both frontend and backend structure for real-world usage.

---

## 📌 Features

* 🏫 Modern responsive homepage
* 🔐 Authentication system (Login/Register)
* 🧑‍💼 Admin panel for managing content
* 📢 Announcements & notices
* 📂 File upload system (notes, resources, etc.)
* 📍 Google Maps integration
* ⚡ Interactive UI with JavaScript

---

## 🛠️ Technologies Used

* HTML5, CSS3, JavaScript
* Backend (PHP)
* Database (phpMyadmin)

---

## 📁 Project Structure


college-website/
│
├── admin/        # Admin dashboard & controls
├── auth/         # Login & authentication system
├── js/           # JavaScript files
├── public/       # Public accessible files (main UI)
├── uploads/      # Uploaded files (notes, images, etc.)
├── LICENSE
```
